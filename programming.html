It all started when some friends at my elementary school started a coding club with the help of our Computer Lab teacher. I joined because it was cool, my friends and I were all doing it together, and because I had the typical fantasy of a kid my age who liked video games and wanted to make them. I actually learned JavaScript back then from Khan Academy, and while I remember a good deal of it now, I can't say I've revisited it. It did, however, set up a base early on for me to have ease learning other languages down the line, and I remember certain aspects like color-picking.


My favorite text editor is Vim. For HTML and CSS I use Atom.

This Website

I coded this website from scratch, learning HTML and CSS in what I consider a short time (one day) and improving my skills since then.
